
# תיקון שגיאות כתיב ויצירת דף ניהול

## סיכום
תיקון כל המקומות שכתוב "נהלים" ל"נחלים" ויצירת דף ניהול (Admin) חסר שגורם לשגיאת 404.

---

## שינויים נדרשים

### 1. תיקון שגיאות כתיב "נהלים" → "נחלים"

| קובץ | שורה | לפני | אחרי |
|------|------|------|------|
| `index.html` | 8 | `כדורגל נהלים` | `כדורגל נחלים` |
| `index.html` | 9 | `בנהלים` | `בנחלים` |
| `index.html` | 15 | `כדורגל נהלים` | `כדורגל נחלים` |
| `index.html` | 18 | `כדורגל נהלים` | `כדורגל נחלים` |
| `index.html` | 19 | `בנהלים` | `בנחלים` |
| `src/pages/Login.tsx` | 104 | `כדורגל נהלים` | `כדורגל נחלים` |
| `src/pages/Dashboard.tsx` | 16 | `כדורגל נהלים` | `כדורגל נחלים` |
| `src/pages/Dashboard.tsx` | 50 | `תושב נהלים` | `תושב נחלים` |
| `src/pages/Profile.tsx` | 110 | `תושב נהלים` | `תושב נחלים` |

### 2. יצירת דף ניהול חדש

**קובץ חדש:** `src/pages/Admin.tsx`

דף ניהול בסיסי עם:
- כותרת וניווט חזרה
- כרטיסים לניהול:
  - ניהול משתמשים (בקרוב)
  - הגדרות אפליקציה (בקרוב)
  - יצירת QR (בקרוב)
  - עריכת חוקים (בקרוב)

### 3. הוספת נתיב Admin ב-App.tsx

```text
Route: /admin
Element: <ProtectedRoute requireAdmin><Admin /></ProtectedRoute>
```

---

## פרטים טכניים

### קבצים לעריכה
1. `index.html` - תיקון 5 מופעים של "נהלים"
2. `src/pages/Login.tsx` - תיקון מופע אחד
3. `src/pages/Dashboard.tsx` - תיקון 2 מופעים
4. `src/pages/Profile.tsx` - תיקון מופע אחד
5. `src/App.tsx` - הוספת import ו-route לדף Admin

### קובץ חדש
`src/pages/Admin.tsx` - דף ניהול עם מבנה דומה לשאר הדפים:
- Header עם כפתור חזור
- רשימת כרטיסים לפונקציות ניהול
- Bottom navigation
- כל הפונקציות יהיו "בקרוב" לעת עתה (placeholder)
